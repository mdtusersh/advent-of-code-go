# advent-of-code-go

My [Advent of Code](https://adventofcode.com/) solutions in Go.

## Project Structure

```
├── 2015/
│   └── day01/
│       ├── main.go        # Solution
│       ├── main_test.go   # Tests with example inputs
│       └── input.txt      # Puzzle input (gitignored)
├── util/
│   └── util.go            # Shared helpers (input parsing, etc.)
└── go.mod
```

## Running Solutions

```sh
# Run a solution
cd 2015/day01
go run .

# Run tests
cd 2015/day01
go test .
```
